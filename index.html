APG Tag Test Page

This page is designed to test your GTM tag without a real GAM setup. It includes mock GPT that can trigger the same events your tag listens for.

1) Mock GPT Controls

Use these to simulate each issue type and confirm you see requests in Hoppscotch and rows in your Sheet.

Trigger Issues

Tip: open DevTools Console to see local logs from the mock. Your tag should still POST to your API endpoint.

3) Expected Results

  • gpt_not_loaded — click “Disable GPT”, then refresh the page.
  • slot_not_registered — enable mock GPT, then click “Trigger: slot_not_registered”.
  • wrong_network — enable mock GPT, then click “Trigger: wrong_network”.
  • empty_response — enable mock GPT, then click “Trigger: empty_response”.
  • unexpected_size — enable mock GPT, then click “Trigger: unexpected_size”.

Your tag dedupes events for 24h by default; if you don't see repeat events, clear localStorage for this site.
